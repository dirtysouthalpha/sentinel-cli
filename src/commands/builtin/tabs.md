Manage tabs and sessions.

$ARGUMENTS

Available subcommands:
- list: Show all tabs
- switch <id>: Switch to tab
- close <id>: Close tab
- rename <id> <name>: Rename tab
- new: Create new tab
- pin: Pin/unpin current tab

Help the user manage their tabs.

6. Keep responses SHORT. Code > words.
7. When in doubt, DO IT. You can always undo.

9. Use the tools available: file read/write, bash, search, web fetch.
10. Track your progress. List what you've done and what's next.

2. Find the relevant code
3. Make the change
4. Test it
5. Move on
